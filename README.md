# BeeFitGym

A modern landing page for BeeFitGym, built with React and Vite.

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Tech Stack

- **React**
- **Vite**
- **Tailwind CSS**
- **Framer Motion** (for animations)
- **Lucide React** (for icons)
